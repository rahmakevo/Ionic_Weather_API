# iWeather

Sample tutorial for Ionic Framework

## Getting started

One will need Node.js installed on your machine
Install Ionic
Install Cordova
Use tabs Ionic Template

### Dependencies and Framework

```
Node.js
Ionic
Ionic-Tabs Template
Underground Weather API
```
